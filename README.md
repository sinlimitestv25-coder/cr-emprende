# C&R Gestión Emprendedores

Base inicial React + Vite + Tailwind para una plataforma de gestión de emprendimientos locales.

## Concepto
- Administrador: crea usuarios, emprendimientos, rubros, módulos, planes, suscripciones y soporte.
- Emprendedor: gestiona productos, stock, insumos, proveedores, recetas, clientes, presupuestos, finanzas y WhatsApp.
- Soporte remoto: el usuario puede habilitar acceso temporal al administrador.

## Ejecutar en local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Despliegue recomendado
- Subir a GitHub.
- Conectar repositorio a Vercel.
- Framework: Vite.
- Build command: npm run build.
- Output directory: dist.
