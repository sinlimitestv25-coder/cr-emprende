# C&R Emprende

Plataforma de gestión para emprendimientos de C&R Soluciones Digitales.

## Incluye

- Branding C&R Soluciones Digitales.
- Fondo SaaS azul en login y banners.
- Dashboard administrador.
- Panel del emprendedor.
- Modal de bienvenida para emprendedores.
- Mensajes internos entre emprendedor y administrador.
- Rubros, módulos, planes, suscripciones y soporte remoto.

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Deploy en Vercel

- Framework: Vite
- Build command: npm run build
- Output directory: dist
